## Project setup

### 1. Install Repository

```
git clone git@github.com:JonAmparo/api-sort-and-search.git
```

### 2. cd into your project directory

```
cd api-sort-and-search
```

### 3. Install dependencies

```
npm install
```

### 4. Launch project in your local enviornment

```
npm run start
```
